# 统一错误码说明

所有接口统一返回：

```json
{ "code": 200, "message": "success", "data": {} }
```

当业务失败时，HTTP 状态码通常保持 200，业务错误通过 `code` 和 `message` 表达；认证失败、权限不足、网关级异常等场景可直接返回对应 HTTP 状态码。

## 通用错误码

| 错误码 | 含义 | 常见 HTTP 状态码 | 前端处理建议 |
|---|---|---|---|
| 200 | 成功 | 200 | 正常读取 `data` |
| 400 | 请求参数错误 | 400/200 | 显示 `message`，提示用户检查输入 |
| 401 | 未登录或 Token 过期 | 401 | 清理登录态并跳转登录页 |
| 403 | 权限不足 | 403 | 提示无权限；若是登录失效则重新登录 |
| 404 | 资源不存在 | 404/200 | 提示记录不存在或已删除 |
| 500 | 系统内部错误 | 500 | 提示稍后重试，并查看后端日志 |

## 认证模块：1001-1099

| 错误码 | 含义 | 前端处理建议 |
|---|---|---|
| 1001 | 用户名或密码错误 | 提示重新输入 |
| 1002 | 账号不存在 | 提示注册或检查账号 |
| 1003 | 账号已被禁用 | 提示联系管理员 |
| 1004 | Token 已过期 | 清理登录态并跳转登录页 |
| 1005 | Token 无效 | 清理登录态并跳转登录页 |
| 1006 | 账号已存在 | 提示更换用户名或手机号 |

## 事项模块：2001-2099

| 错误码 | 含义 | 触发场景 |
|---|---|---|
| 2001 | 事项不存在 | 提交申请或查询事项时 `itemId` 无效 |
| 2002 | 事项已下线 | 事项状态不是 `online` |
| 2003 | 不满足申请条件 | 后续接入条件校验时使用 |

## 申请模块：2101-2199

| 错误码 | 含义 | 触发场景 |
|---|---|---|
| 2101 | 申请单不存在 | 查询详情、补件、撤回、上传材料时申请不存在 |
| 2102 | 申请状态不允许该操作 | 非待补件/已撤回状态重新提交，或不允许撤回的状态执行撤回 |
| 2103 | 无权操作该申请 | 非申请人或授权代理人访问申请 |

## 工单模块：2201-2299

| 错误码 | 含义 | 触发场景 |
|---|---|---|
| 2201 | 工单不存在 | 查询或审核不存在的工单 |
| 2202 | 无权处理该工单 | 后续按工作人员归属限制时使用 |
| 2203 | 工单状态不允许该操作 | 审核动作不在允许范围内 |

## 预约模块：2301-2399

| 错误码 | 含义 | 触发场景 |
|---|---|---|
| 2301 | 服务暂不可用 | 预约的服务资源未开放 |
| 2302 | 预约时间冲突 | 同一用户或资源时间段冲突 |
| 2303 | 预约记录不存在 | 查询或更新预约时记录不存在 |
| 2304 | 预约状态不允许该操作 | 已取消/已完成预约再次取消等 |

## 材料相关通用校验

当前材料模块复用通用错误码：

| 场景 | 错误码 | message 示例 |
|---|---|---|
| 材料名称为空 | 400 | 材料名称不能为空 |
| 材料模板不存在 | 404 | 材料模板不存在 |
| 材料模板不属于当前申请事项 | 400 | 材料模板不属于当前申请事项 |
| 材料预审状态非法 | 400 | 材料预审状态错误 |
| 申请材料不存在 | 404 | 申请材料不存在 |
| 审核通过/办结时必填材料缺失 | 400 | 必填材料不完整，缺少：居住证明材料 |
| 上传文件为空 | 400 | 上传文件不能为空 |
| 上传文件过大 | 400 | 上传文件不能超过 20MB |
| 上传格式不支持 | 400 | 文件格式不支持，仅支持 PDF、JPG、PNG、DOC、DOCX |
| 文件路径非法 | 400 | 文件路径非法 |
| 材料文件不存在 | 404 | 材料文件不存在 |
| 审核通过/办结时存在预审失败材料 | 400 | 存在预审未通过的材料，不能审核通过或办结 |

说明：当前材料错误仍复用通用错误码。若后续接入对象存储、OCR/AI 预审或病毒扫描，可视复杂度再增加 `2401-2499` 材料专用错误码。
