# 统一错误码说明

## 通用错误码

| 错误码 | 含义 | HTTP 状态码 |
|---|---|---|
| 200 | 成功 | 200 |
| 400 | 请求参数错误 | 400 |
| 401 | 未登录或 Token 过期 | 401 |
| 403 | 权限不足 | 403 |
| 404 | 资源不存在 | 404 |
| 500 | 系统内部错误 | 500 |

## 认证模块（1001-1099）

| 错误码 | 含义 | 前端处理建议 |
|---|---|---|
| 1001 | 用户名或密码错误 | 提示重新输入 |
| 1002 | 账号不存在 | 提示注册 |
| 1003 | 账号已被禁用 | 提示联系管理员 |
| 1004 | Token 已过期 | 重定向登录页 |
| 1005 | Token 无效 | 重定向登录页 |
| 1006 | 账号已存在 | 提示更换用户名 |

## 事项模块（2001-2099）

| 错误码 | 含义 |
|---|---|
| 2001 | 事项不存在 |
| 2002 | 事项已下线 |
| 2003 | 不满足申请条件 |

## 申请模块（2101-2199）

| 错误码 | 含义 |
|---|---|
| 2101 | 申请单不存在 |
| 2102 | 申请状态不允许该操作 |
| 2103 | 无权操作该申请 |

## 工单模块（2201-2299）

| 错误码 | 含义 |
|---|---|
| 2201 | 工单不存在 |
| 2202 | 无权处理该工单 |
| 2203 | 工单状态不允许该操作 |

## 预约模块（2301-2399）

| 错误码 | 含义 |
|---|---|
| 2301 | 服务暂不可用 |
| 2302 | 预约时间冲突 |
| 2303 | 预约记录不存在 |
| 2304 | 预约状态不允许该操作 |
